
# Mockups van shaders + shader collider

Mockup maps van verschillende biotopen die allen een ander deel van shading gebruiken(Lerp, Tiling etc) 
Een collider die een speler over shaded objects die een hightmap hebben kan laten lopen.

## Features
Shader collider.
verschillende mockups van verschillende biotopen.


## Software Anaylse

Unreal engine: Vrij simpel om te leren met een goede tutorial. Node based shading builder is erg makkelijk in het gebruik. Hoewel mist de features die ik moet benutten die bijv. Unity wel heeft 
Unity: Zelf shader schrijven is een goede optie. de kennis die ik beheers is dusdanig dat ik zelfstandig er iets mee kan bouwen. Kan een chaos worden aangezien er veel bij komt kijken. Toch blijft het feit dat ik niet alle ins en outs nog weet.
ShaderGraph: Goede optie. Heeft erg veel nodes om wat mee te doen. Goede documentation. Net als unreal makkelijk in het gebruik. Kan een zooitje worden met alle lijntjes.
